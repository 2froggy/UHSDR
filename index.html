Important information related firmware upgrades!!


If you have mcHF run with firmware <= 0.219.26.15 and want to upgrade to a newer version (>= 0.219.26.16) you loose configuration data because of new bands are added and therefore storage format has changed. Follow these steps
  • write down all your configuration settings *before* you start upgrade
  • upgrade firmware
  • reset setttings at first start (press MENU+SPLIT+TUNE simultaneously and hold them, switch on mcHF and hold buttons until message appears. Then release buttons and switch off mcHF with power button.
  • then start mcHF again
  • now enter manually data you have written down
  • after finishing switch off mcHF with power button and on again
  • if you use serial EEPROM and do havee a backup of your old configuration overwrite it now with menu #197 because old config is incompatible
  • ready...
If this is first firmware you install you have to do nothing specail.
I am sorry for this but if you want to participate in new bands you *MUST* go once this procedure.

mchf is a project of Chris, M0NKA

It is a standalone shortwave SDR amateur radio for homebrewing. It is free for non-commercial use.

There is a Yahoo Newsgroup in English and a German project page of DARC OV I40 managed by DF8OE with a German discussion group for all German-speaking people.

This GitHub wants to support community working on firmware and bootloader code, showing gathered informations of modifications and it contains an easy-to-use bug tracking and feature wish list system.

I hope we all will work together to help each other at the building and bug fixing and improving firmware.

Many thanks Chris for offering such a nice project to the amateur radio world! This project has the power to awake interest to get a callsign and bring young people to our beautiful hobby!
